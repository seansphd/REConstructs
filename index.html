A practical study in preservation by recreation

The project tests whether a systems thinking frame can help language models recreate historical computer art with better experiential fidelity. Ten works from the Computer Arts Society’s PAGE magazine form the testbed. Two approaches are compared. A direct implementation from the original text. A two stage systems led version that restructures constants, variables, interaction, feedback, and emergence before coding.

Abstract

The study compares two pathways for recreating historic algorithms. A direct reading of artists’ descriptions, and a route that first distils systems features before any code is written. Systems led builds show higher alignment with conceptual and interactive aims, while direct builds show sharper mathematical accuracy. The work argues for balancing precision with the feel of the original practice when presenting archives to the public.

Method

  • Ten PAGE projects selected for depth of description and feasibility.
  • Two implementations per project using the same model and target stack.
  • Three forms of analysis. System diagrams. Code to text comparison. Researcher review of learning value.
